## 🚀 البدء السريع

### 1. التثبيت على Vercel (الطريقة الأسهل)

1. انسخ هذا المشروع إلى GitHub
2. اذهب إلى [vercel.com](https://vercel.com)
3. اضغط "Import Project"
4. اختر المشروع من GitHub
5. اضغط Deploy

### 2. التشغيل المحلي

```bash
# تثبيت المتطلبات
npm install

# تشغيل خادم التطوير
npm run dev

# سيفتح على http://localhost:3000
```

### 3. البناء والنشر

```bash
# بناء المشروع
npm run build

# تشغيل النسخة الإنتاجية
npm start
```

## 🎨 واجهة المستخدم

### ميزات الواجهة:
- **رؤية شاملة**: عرض كل المهام في صفحة واحدة
- **تصفية ديناميكية**: اختر الفئة المطلوبة
- **شريط تقدم**: تابع نسبة الإنجاز
- **ملاحظات سريعة**: أضف ملاحظات لأي مهمة
- **ألوان ذكية**: رموز بصرية للأولويات

## 💾 حفظ البيانات

البيانات تُحفظ بطريقتين:
1. **محلياً**: في متصفح الجهاز (localStorage)
2. **على الخادم**: عند الاتصال بالإنترنت

هذا يعني أن البيانات آمنة حتى بدون إنترنت!

## 👥 المشاركة مع الفريق

1. انسخ رابط الموقع

3. كل شخص يمكنه:
   - تحديث حالة المهام
   - إضافة ملاحظات
   - متابعة التقدم

## 📤 تصدير البيانات

اضغط زر "تصدير" لتنزيل نسخة JSON من كل البيانات

## 🔧 التخصيص

### لتغيير المهام الافتراضية:
عدّل قائمة `defaultTasks` في `pages/index.js`

### لتغيير الألوان:
عدّل `tailwind.config.js`

### لإضافة فئات جديدة:
أضفها في مصفوفة `categories` وفي البيانات الافتراضية
